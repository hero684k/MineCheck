❓ Частые вопросы

Это безопасно? Антивирус ругается+

Да. Утилиты детектятся как RiskTool — это ложные срабатывания. Добавьте папку в исключения.

Нужен ли Python?+

Нет! Всё в одном .exe. Просто запустите Minecheck.exe.

Работает на Windows 7?+

Рекомендуется Windows 10/11. На Windows 7 часть инструментов может не работать.

Можно добавить свои инструменты?+

Да! Проект открыт (MIT). Скачайте исходный код на GitHub.
