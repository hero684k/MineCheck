❓ Частые вопросы

Это безопасно? Антивирус ругается+

Да, безопасно. Утилиты NirSoft и Process Hacker детектятся как RiskTool/HackTool, потому что могут использоваться для анализа системы. Это ложные срабатывания. Добавьте папку в исключения антивируса.

Нужен ли Python для запуска?+

Нет! Всё упаковано в один .exe файл. Просто запустите Minecheck.exe.

Работает на Windows 7?+

Рекомендуется Windows 10/11. На Windows 7 некоторые инструменты могут не работать.

Можно ли добавить свои инструменты?+

Да! Проект открыт (MIT License). Скачайте исходный код на GitHub и добавьте свои .exe файлы.
